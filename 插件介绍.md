# GameFunXiao 插件介绍

## 简介

GameFunXiao 是一个基于 Paper 1.21.11 的 Minecraft 猎人游戏插件。玩家可以创建或加入房间，进行猎人追逐猎物的游戏。猎物需要在规定时间内完成目标（击败末影龙或存活一定时间），猎人则需要在此之前击杀猎物。

---

## 游戏模式

| 模式 | ID | 通关目标 | 特点 | 猎人奖励 | 猎物奖励 |
|------|----|----------|------|----------|----------|
| 经典模式 | `classic` | 击败末影龙 | 猎人有追踪指南针，连续扔掉两次可获得一次TP机会 | 150 | 300 |
| 随机指南针 | `random_compass` | 击败末影龙 | 每5分钟随机一位玩家获得特殊指南针，使用后敌方获得发光+缓慢+饥饿 | 150 | 300 |
| 互换模式 | `swap` | 击败末影龙 | 需要两位猎物，每1分钟两位猎物互换背包和位置 | 300 | 400 |
| 无有模式 | `no_item` | 击败末影龙 | 没有指南针，没有TP，全凭实力追踪 | 500 | 600 |
| 存活模式 | `survival` | 存活30分钟 | 猎人有追踪指南针，猎物需要躲避追杀 | 50 | 100 |

---

## 游戏流程

```
玩家加入房间
    ↓
等待大厅（等待其他玩家）
    ↓
人数达到最少人数后开始倒计时（默认5分钟）
    ↓
投票选出猎物（或随机选择）
    ↓
猎物进入游戏世界选择世界（2分钟）
    ↓
猎物按下开始按钮，3秒倒计时
    ↓
游戏正式开始
    ↓
猎物完成目标 → 猎物胜利
猎人击杀猎物 → 猎人胜利
    ↓
发放奖励 + 显示本局排行榜
    ↓
倒计时结束后返回原位置
```

## 命令列表

### 玩家命令

| 命令 | 说明 |
|------|------|
| `/gamefunxiao` | 打开主菜单 |
| `/gamefunxiao menu` | 打开主菜单 |
| `/gamefunxiao help` | 查看命令帮助 |
| `/gamefunxiao huntergame` | 打开猎人游戏菜单 |
| `/gamefunxiao hg` | 打开猎人游戏菜单（简写） |
| `/gamefunxiao leave` | 离开当前房间 |
| `/gamefunxiao quit` | 离开当前房间（别名） |
| `/gamefunxiao rejoin` | 重新加入游戏（猎人断线后使用） |

### 管理员命令

| 命令 | 说明 | 权限 |
|------|------|------|
| `/gamefunxiao reload` | 重载配置文件 | `gamefunxiao.admin` |
| `/gamefunxiao setlobbyspawn` | 设置大厅出生点 | `gamefunxiao.admin` |
| `/gamefunxiao hg create <模式> <人数> [修饰符...]` | 创建自定义房间 | `gamefunxiao.admin` |

### 创建房间示例

```
/gamefunxiao hg create classic 8
/gamefunxiao hg create survival all -ThunderStorm -RewardChest
```

## 修饰符列表

修饰符可以在创建房间时添加，格式为 `-修饰符名`。

| 修饰符 | 说明 |
|--------|------|
| `ThunderStorm` | 开启雷暴天气 |
| `RewardChest` | 地图上生成奖励箱 |
| `NoWorld` | 禁止换世界 |
| `YesYChange` | 允许Y轴变化 |
| `YesHunterSee` | 猎人开始时不会致盲 |
| `IncludeY` | 指南针距离包含Y轴 |
| `HunterDropItems` | 猎人死亡掉落物品 |
| `PreyRespawn` | 猎物可以复活 |
| `HunterTPOnDeath` | 猎人死亡后获得TP机会 |

## 配置文件说明

### config.yml

```yaml
# 消息前缀（通用）
prefix: "§x§F§F§6§6§0§0ManHunt §x§D§D§4§4§0§0>> §f"

# 猎人游戏消息前缀
hunter_game_prefix: "§x§F§F§6§6§0§0⚔ §x§F§F§9§9§3§3猎人游戏 §x§F§F§6§6§0§0» §f"

# 游戏设置
game:
  min_players: 2          # 最少开始人数
  wait_countdown: 300     # 等待倒计时（秒）
  world_selection_time: 120  # 世界选择时间（秒）
  survival_time: 30       # 存活模式时间（分钟）
  one_vs_one_quit_penalty: true  # 1v1退出惩罚

# 聊天设置
chat:
  isolate_game_chat: true       # 隔离游戏世界聊天
  notify_spectator_join: true   # 通知旁观者加入
  admin_spectate_notify: false  # 管理员旁观是否通知

# 奖励设置
rewards:
  fail_reward: 25  # 失败时猎物获得的金币

# 自定义金币设置
custom_currency:
  enabled: false
  name: "金币"
  take_command: "/eco take %player% %amount%"
  give_command: "/eco give %player% %amount%"
  balance_placeholder: "%vault_eco_balance%"
```

### messages.yml

所有聊天框消息都可以在 `messages.yml` 中自定义，支持以下变量：

| 变量 | 说明 |
|------|------|
| `{player}` | 玩家名 |
| `{room_id}` | 房间ID |
| `{mode}` | 游戏模式名 |
| `{current}` | 当前人数 |
| `{max}` | 最大人数 |
| `{time}` | 时间 |
| `{amount}` | 金币数量 |
| `{target}` | 目标玩家名 |

## 菜单系统

插件内置完整的菜单系统，所有操作均可通过菜单完成：

- **主菜单** - 插件入口，显示所有功能
- **猎人游戏导航菜单** - 选择游戏模式或查看房间
- **房间列表菜单** - 查看所有当前房间，可加入或旁观
- **创建房间菜单** - 自定义创建房间
- **投票猎物菜单** - 在等待大厅投票选择猎物
- **排行榜菜单** - 查看各类排行榜（天榜/周榜/年榜/总榜）
- **传送队友菜单** - 猎人死亡后传送到队友身边

## 游戏内物品

### 等待大厅物品

| 槽位 | 物品 | 功能 |
|------|------|------|
| 0 | 投票猎物 | 右键打开投票菜单 |
| 4 | 强制开始（管理员） | 右键强制开始游戏 |
| 7 | 宣传房间 | 右键发送宣传消息（30秒冷却） |
| 8 | 退出游戏 | 右键退出房间 |

### 世界选择物品（猎物）

| 槽位 | 物品 | 功能 |
|------|------|------|
| 3 | 选择当前世界 | 右键确认选择此世界 |
| 5 | 换一个世界 | 右键重新生成世界 |

### 游戏中物品（猎人）

| 物品 | 功能 |
|------|------|
| 追踪指南针 | 主手或副手持有，显示最近猎物距离，连续扔掉两次获得TP机会 |

## 排行榜系统

排行榜记录以下数据：

- **猎物胜利次数** - 作为猎物成功通关的次数
- **猎人胜利次数** - 作为猎人成功击杀猎物的次数
- **游玩次数** - 总游玩次数
- **最快通关时间** - 作为猎物的最快通关时间

支持时间维度：**天榜 / 周榜 / 年榜 / 总榜**

## 注意事项

1. 自定义房间（非管理员创建）不计入排行榜，也不发放奖励
2. 猎人断线后有5分钟时间重新加入，使用 `/gamefunxiao rejoin` 重连
3. 1v1模式下（1猎人1猎物），任意一方退出游戏将不发放奖励
4. 游戏结束后会自动清理游戏世界，玩家返回原来的位置
5. 执行 `reload` 命令时会自动检测并补全缺失的配置文件
